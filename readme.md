My CV, build with LaTex and Gulp
--------------------------------

- To user `gulp serve`, you will need a non-blocking pdf viewer.
For example: Sumatra on Window, zathura on linux.


Note
----

- Use `texliveonfly` to quickly install missing LaTex package.
